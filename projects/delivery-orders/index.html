Delivery Orders

Project: Web Application for Managing Company Purchases
A web-based application designed to manage delivery orders within an organization. The system enables users to create and manage delivery orders efficiently. It integrates with a thermal printer to print physical delivery slips, which are used to authorize employees to load products onto designated vehicles. The application supports multiple user roles, allowing different levels of access and permissions based on responsibilities. Some users can create and manage delivery orders, while others have permissions to export them — ensuring a smooth and secure workflow from order creation to dispatch.

Key Features:

  • Multi-user access with role-based permissions
  • Create, manage, and export delivery orders
  • Thermal printer integration for printing delivery slips
  • Intuitive, user-friendly interface

Tools & Technologies:

  • Python (Django Framework)
  • sqlite
  • HTML, CSS, JavaScript, jQuery
  • Shell Scripting
  • Git & GitHub
